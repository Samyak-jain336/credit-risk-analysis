# Credit Risk Analysis Dashboard

This project presents an interactive Power BI dashboard designed to analyze customer credit risk using financial and demographic data.

## Key Features
- KPI cards for customer count, credit score, income, and debt-to-income ratio
- Default vs non-default customer segmentation
- Income vs debt-to-income scatter analysis to identify high-risk clusters
- Default rate analysis by age group using DAX measures

## Tools Used
- Power BI
- DAX
- Data Visualization
